# Week 05：机械臂逆运动学控制与视觉传感器仿真

## 1. 实验目标
- 在 PyBullet 环境中实现对 6 自由度（DoF）机械臂的精准控制。
- 掌握**逆运动学 (Inverse Kinematics, IK)** 的计算原理，使末端执行器达到指定空间位姿。
- 学习配置并读取仿真环境中的虚拟相机数据（RGB、深度图、分割图）。

## 2. 实验环境
- **操作系统**: Ubuntu 24.04 LTS
- **物理引擎**: PyBullet
- **核心组件**: Panda Arm URDF, Synthetic Camera

## 3. 实验内容与步骤

### 3.1 机械臂逆运动学 (IK)
通过调用 PyBullet 的核心算法接口，输入目标笛卡尔坐标，计算出各关节的旋转角度。
- **目标位姿设定**: 实验中将机械臂末端精准定位至桌面物体上方。
- **关节控制**: 通过 `setJointMotorControlArray` 实现多关节同步运动。

### 3.2 视觉传感器仿真 (Computer Vision)
实验中配置了三个子窗口实时显示相机反馈：
1. **RGB data**: 彩色画面，用于基础环境识别。
2. **Depth data**: 深度信息，用于感知物体距离和空间建模。
3. **Segmentation mask**: 语义分割，用于将机械臂与背景环境进行像素级分离。
