# Week 08：基于 Docker 容器的 ROS2 环境部署与小乌龟仿真

## 1. 实验目标
- 学习使用 **Docker** 容器化技术部署一致性的机器人开发环境。
- 掌握通过浏览器访问容器内 GUI 界面（基于 NoVNC）的方法。
- 在 Docker 镜像中验证 ROS2 环境的完整性并运行 `turtlesim` 节点。

## 2. 实验环境
- **宿主机 OS**: Ubuntu 24.04 LTS
- **虚拟化技术**: Docker (Linux Containers)
- **访问协议**: HTTP / NoVNC (端口 6080)
- **机器人框架**: ROS2 (预安装于镜像内)

## 3. 实验内容与步骤

### 3.1 容器启动与连接
通过 Docker 启动包含 ROS2 环境的镜像，并映射本地端口。在浏览器地址栏输入：
`http://127.0.0.1:6080/`
连接成功后，即可看到集成在容器内的 Ubuntu 桌面环境。
