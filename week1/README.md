# Week 01 - 环境准备与工具安装

## 1. 作业说明

本周主要完成课程仓库的基础准备工作，包括 GitHub 账号、VS Code 编辑器、本地 Git 环境和 SSH 连接配置，为后续每周作业提交建立稳定流程。

## 3. 实验环境

- GitHub
- VS Code
- Git
- SSH Key

## 4. 实验步骤

1. 注册并配置 GitHub 账号。
2. 安装 VS Code 并熟悉 Markdown 编辑。
3. 配置 Git 用户信息。
4. 生成 SSH Key 并连接 GitHub。

## 5. 运行命令

<pre><code class="language-bash">
git config --global user.name your-name
git config --global user.email your-email@example.com
ssh-keygen -t ed25519
git status
</code></pre>
